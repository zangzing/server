Download ZangZing

1. Downloading the photo uploader

Your ZangZing download should start automatically within a few seconds. If it doesn't, click here.

2. Run the ZangZing Installer
Click Save to download and then go to Tools menu, choose Downloads and then double click ZangZing.exe
Click Run to start the installer, then Yes to allow ZangZing to make changes.
Click the ZangZing.exe file that is in the lower left corner of your window.
In the Downloads window, double click the ZangZing.pkg file.
3. Click Continue to install
Follow the instructions.
4. It's ready when you see ZangZing
When you see the logo, you are ready to go and upload photos!
